============
Installation
============

At the command line::

    $ pip install awsapicli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv awsapicli
    $ pip install awsapicli

Or, if you are using pipenv::

    $ pipenv install awsapicli

Or, if you are using pipx::

    $ pipx install awsapicli
